# dualACOPFsolver
Implementation of a proximal bundle method (PBM) to solve the ACOPF's dual problem.
In particular, we use this PBM as a processing step to polish MOSEK's dual solution.

# Dependencies

Required python packages:
- numpy
- scipy
- pandas
- osqp
- chompack
